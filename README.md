# Student Record System in Python

This project demonstrates how a school can save and manage student records in Python.
It includes multiple concepts such as lists, OOP (Object-Oriented Programming), abstraction, encapsulation, inheritance, polymorphism, and decorators.

This repository is perfect for beginners who want to learn Python and understand how real-world objects can be modeled using classes and objects.


# **Features Covered**

1. Student Records Using Lists
Basic storage of student details using Python lists
Simple printing and formatting using f-strings
